# 🏠 House Price Prediction using Linear Regression

A Machine Learning project that predicts house prices based on various house features using **Linear Regression** and **Scikit-learn**.

---

## 📌 Project Overview

This project uses a housing dataset to train a Linear Regression model capable of predicting house prices based on features such as:

- Area
- Number of Bedrooms
- Number of Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

The project also allows users to enter details of a new house and receive a predicted house price.

---

## 🚀 Features

- Load housing dataset
- Data preprocessing using Label Encoding
- Train/Test Split
- Linear Regression Model
- Model Evaluation using R² Score
- Compare Actual vs Predicted Prices
- Predict House Price using User Input

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn

---

## 📂 Dataset

Housing Dataset from Kaggle.

---


## 📈 Model Performance

- Algorithm: Linear Regression
- Evaluation Metric: R² Score
- Predicts house prices based on 12 different features.
